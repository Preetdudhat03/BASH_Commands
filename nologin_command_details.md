# nologin Command - Detailed Documentation

## 1. Description
The `nologin` command politely refuses a login. It is set as the shell for accounts that should not log in (like `www-data`).

## 2. Purpose
The purpose of `nologin` is security.

## 3. Examples
### Example 1: Usage
Set `/usr/sbin/nologin` as shell in `/etc/passwd`.
