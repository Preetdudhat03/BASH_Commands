# qpdf Command - Detailed Documentation

## 1. Description
The `qpdf` command is a tool for structural, content-preserving transformations on PDF files.

## 2. Purpose
The purpose of `qpdf` is optimizing/encrypting PDFs.

## 3. Examples
### Example 1: Linearize (Web optimize)
```bash
qpdf --linearize in.pdf out.pdf
```
