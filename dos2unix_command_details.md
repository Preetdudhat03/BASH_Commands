# dos2unix Command - Detailed Documentation

## 1. Description
The `dos2unix` command converts text file line endings from DOS/Mac format (CRLF) to Unix format (LF).

## 2. Purpose
The purpose of `dos2unix` is compatibility. Fix scripts written on Windows that fail on Linux.

## 3. Examples
### Example 1: Fix File
```bash
dos2unix script.sh
```
