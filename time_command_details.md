# time Command - Detailed Documentation

## 1. Description
The `time` command measures how long a command takes to execute.

## 2. Purpose
The purpose of `time` is benchmarking. It is used to:
- Compare the speed of two different scripts.

## 3. Output
- **real**: Wall clock time (actual elapsed time).
- **user**: CPU time in user mode.
- **sys**: CPU time in kernel mode.

## 4. Examples
### Example 1: Time a Command
```bash
time sleep 2
```
