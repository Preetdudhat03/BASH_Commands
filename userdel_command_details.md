# userdel Command - Detailed Documentation

## 1. Description
The `userdel` command deletes a user account and related files.

## 2. Purpose
The purpose of `userdel` is account removal. It is used to:
- Remove access for a user.

## 3. Parameters
- `-r`: Remove home directory and mail spool.

## 4. Examples
### Example 1: Remove User
```bash
sudo userdel -r olduser
```
