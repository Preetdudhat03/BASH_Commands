# expand Command - Detailed Documentation

## 1. Description
The `expand` command converts tabs into spaces. The `unexpand` command does the reverse.

## 2. Purpose
The purpose of `expand` is indentation fixing. It is used to:
- Ensure code alignment on different editors.

## 3. Examples
### Example 1: Tabs to Spaces
Convert tabs to 4 spaces.
```bash
expand -t 4 code.py
```
