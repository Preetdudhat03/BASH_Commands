# bg Command - Detailed Documentation

## 1. Description
The `bg` command resumes suspended jobs (e.g., stopped with Ctrl+Z) in the background.

## 2. Purpose
The purpose of `bg` is multitasking. It is used to:
- Continue a large copy command while getting the terminal prompt back.

## 3. Examples
### Example 1: Background Job
Start a job, press Ctrl+Z, then run:
```bash
bg
```
