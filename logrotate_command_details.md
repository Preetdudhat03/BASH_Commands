# logrotate Command - Detailed Documentation

## 1. Description
The `logrotate` command is designed to simplify administration of systems that generate large numbers of log files.

## 2. Purpose
The purpose of `logrotate` is rotating, compressing, and mailing logs.

## 3. Examples
### Example 1: Force Rotate
```bash
sudo logrotate -f /etc/logrotate.conf
```
