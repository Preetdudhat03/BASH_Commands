# tree Command - Detailed Documentation

## 1. Description
The `tree` command lists the contents of directories in a tree-like format. It provides a visual depth structure. *Note: Usually needs to be installed via package manager.*

## 2. Purpose
The purpose of `tree` is visualization. It is used to:
- Understand project structure.
- Generate documentation for folder hierarchies.

## 3. Parameters
- `-L [level]`: Max display depth.
- `-d`: List directories only.

## 4. Examples
### Example 1: Depth 2
```bash
tree -L 2
```
