# btrfs-image Command - Detailed Documentation

## 1. Description
The `btrfs-image` command creates an image of a Btrfs filesystem.

## 2. Purpose
The purpose of `btrfs-image` is dumping metadata for debugging.

## 3. Examples
### Example 1: Create Image
```bash
sudo btrfs-image /dev/sda1 image.dump
```
