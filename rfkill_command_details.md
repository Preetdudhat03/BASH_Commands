# rfkill Command - Detailed Documentation

## 1. Description
The `rfkill` command queries the state of, and enables/disables, wireless devices (Wi-Fi, Bluetooth).

## 2. Purpose
The purpose of `rfkill` is hardware swtiching.
- Soft-block Wi-Fi (Airplane mode).

## 3. Examples
### Example 1: Block All
```bash
sudo rfkill block all
```
