# mkswap Command - Detailed Documentation

## 1. Description
The `mkswap` command sets up a Linux swap area on a device or in a file.

## 2. Purpose
The purpose of `mkswap` is swap creation. It is used to:
- Format a partition or file to be used as virtual memory (swap).

## 3. Examples
### Example 1: Make Swap
```bash
sudo mkswap /dev/sdb2
```
