# truncate Command - Detailed Documentation

## 1. Description
The `truncate` command shrinks or extends the size of a file to the specified size.

## 2. Purpose
The purpose of `truncate` is file sizing. It is used to quickly clear a log file (`-s 0`).

## 3. Examples
### Example 1: Clear Log
```bash
truncate -s 0 app.log
```
