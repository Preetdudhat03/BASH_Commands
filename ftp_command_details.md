# ftp Command - Detailed Documentation

## 1. Description
The `ftp` command is the standard File Transfer Protocol client. **Note:** FTP is unencrypted and generally insecure.

## 2. Purpose
The purpose of `ftp` is legacy transfer. It is used to:
- Connect to old FTP servers to download/upload files.

## 3. Examples
### Example 1: Connect
```bash
ftp ftp.example.com
```
