# clear Command - Detailed Documentation

## 1. Description
The `clear` command clears the terminal screen. It scrolls the current output out of view, giving you a fresh, clean prompt at the top.

## 2. Purpose
The purpose of `clear` is visibility. It is used to:
- Remove clutter before running a new command.
- Hide previous sensitive output (visually, though it remains in history).

## 3. Shortcut
- **Ctrl+L**: This keyboard shortcut performs the same action as typing `clear`.

## 4. Examples

### Example 1: Clear Screen
```bash
clear
```
