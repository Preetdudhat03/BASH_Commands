# sftp Command - Detailed Documentation

## 1. Description
The `sftp` command stands for "Secure File Transfer Protocol". It runs over SSH.

## 2. Purpose
The purpose of `sftp` is secure transfer. It is used to:
- Upload/download files safely to any server running SSH.

## 3. Examples
### Example 1: Connect
```bash
sftp user@192.168.1.50
```
