# nano Command - Detailed Documentation

## 1. Description
The `nano` command converts the terminal into a simple, easy-to-use text editor. Unlike `vim` or `emacs`, it has a low learning curve and displays available shortcuts at the bottom of the screen.

## 2. Purpose
The purpose of `nano` is editing. It is used to:
- Quickly edit config files.
- Write simple scripts.

## 3. Usage
- **Ctrl+O**: Save (Write Out).
- **Ctrl+X**: Exit.
- **Ctrl+W**: Search (Where Is).

## 4. Examples

### Example 1: Open File
Edit `index.html`.
```bash
nano index.html
```

### Example 2: Line Number
Open file at line 20.
```bash
nano +20 script.py
```
