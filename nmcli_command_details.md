# nmcli Command - Detailed Documentation

## 1. Description
The `nmcli` command is a command-line tool for controlling NetworkManager.

## 2. Purpose
The purpose of `nmcli` is network config. It is used to:
- Connect to Wi-Fi from the terminal.
- Set static IPs.

## 3. Examples
### Example 1: Show Connections
```bash
nmcli connection show
```
