# mkinitrd Command - Detailed Documentation

## 1. Description
The `mkinitrd` command creates an initial ramdisk image used for preloading modules.

## 2. Purpose
The purpose of `mkinitrd` is creating boot images (legacy/RedHat).

## 3. Examples
### Example 1: Create
```bash
mkinitrd /boot/initrd-custom.img 3.10.0
```
