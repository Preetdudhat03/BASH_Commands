# lscpu Command - Detailed Documentation

## 1. Description
The `lscpu` command displays information about the CPU architecture.

## 2. Purpose
The purpose of `lscpu` is hardware checking. It is used to:
- Check number of cores/threads.
- Check CPU virtualization support (VT-x).

## 3. Examples
### Example 1: CPU Info
```bash
lscpu
```
