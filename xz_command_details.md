# xz Command - Detailed Documentation

## 1. Description
The `xz` command is a general-purpose data compression tool with high compression ratio (LZMA). Files end in `.xz`.

## 2. Purpose
The purpose of `xz` is maximum compression.

## 3. Examples
### Example 1: Compress
```bash
xz file.tar
```
### Example 2: Decompress
```bash
xz -d file.tar.xz
```
