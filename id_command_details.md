# id Command - Detailed Documentation

## 1. Description
The `id` command prints real and effective user and group IDs.

## 2. Purpose
The purpose of `id` is verification.
- Check which groups you belong to.
- Verify user ID numbers.

## 3. Examples
### Example 1: Check Self
```bash
id
```
### Example 2: Check User
```bash
id root
```
