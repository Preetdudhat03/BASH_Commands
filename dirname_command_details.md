# dirname Command - Detailed Documentation

## 1. Description
The `dirname` command strips the last component from a file name. It prints the directory path.

## 2. Purpose
The purpose of `dirname` is path extraction. It is used to:
- Get the folder path a file resides in.

## 3. Examples
### Example 1: Get Directory
```bash
dirname /usr/bin/python
# Output: /usr/bin
```
