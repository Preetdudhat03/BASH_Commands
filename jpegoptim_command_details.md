# jpegoptim Command - Detailed Documentation

## 1. Description
The `jpegoptim` command provides lossless optimization (based on optimizing the Huffman tables) and "lossy" optimization on JPEG files.

## 2. Purpose
The purpose of `jpegoptim` is compressing JPEGs.

## 3. Examples
### Example 1: Compress
```bash
jpegoptim image.jpg
```
