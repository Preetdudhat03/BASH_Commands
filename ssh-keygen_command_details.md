# ssh-keygen Command - Detailed Documentation

## 1. Description
The `ssh-keygen` command generates, manages and converts authentication keys for ssh.

## 2. Purpose
The purpose of `ssh-keygen` is creating SSH key pairs for secure passwordless authentication.

## 3. Examples
### Example 1: Generate Key
```bash
ssh-keygen -t ed25519 -C "email@example.com"
```
