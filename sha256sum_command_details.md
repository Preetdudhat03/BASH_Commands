# sha256sum Command - Detailed Documentation

## 1. Description
The `sha256sum` command computes and checks SHA256 message digests. It is more secure than MD5.

## 2. Purpose
The purpose of `sha256sum` is secure verification. It is used to:
- Verify Linux ISO downloads safely.

## 3. Examples
### Example 1: Check Hash
```bash
sha256sum ubuntu.iso
```
