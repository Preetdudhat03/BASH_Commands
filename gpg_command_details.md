# gpg Command - Detailed Documentation

## 1. Description
The `gpg` (GnuPG) command allows you to encrypt and sign your data and communications.

## 2. Purpose
The purpose of `gpg` is encryption. It is used to:
- Encrypt files for specific recipients.
- Verify download signatures.

## 3. Examples
### Example 1: Encrypt File
```bash
gpg -c file.txt
```
