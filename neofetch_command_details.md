# neofetch Command - Detailed Documentation

## 1. Description
The `neofetch` command is a CLI system information tool written in BASH. It displays information about your OS, software and hardware in an aesthetic and visually pleasing way.

## 2. Purpose
The purpose of `neofetch` is showing off system specs and rice.

## 3. Examples
### Example 1: Run
```bash
neofetch
```
