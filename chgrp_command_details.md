# chgrp Command - Detailed Documentation

## 1. Description
The `chgrp` command allows you to change the group ownership of files. Note that `chown` can also do this.

## 2. Purpose
The purpose of `chgrp` is group management. It is used to:
- Change only the group, leaving the owner alone.

## 3. Examples
### Example 1: Change Group
```bash
chgrp staff shared_folder
```
