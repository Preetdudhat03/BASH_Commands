# shopt Command - Detailed Documentation

## 1. Description
The `shopt` command allows you to toggle shell options that change Bash's behavior.

## 2. Purpose
The purpose of `shopt` is customization.
- Enable case-insensitive globbing.
- Enable autocd.

## 3. Examples
### Example 1: Auto CD
Allow changing folders just by typing directory name.
```bash
shopt -s autocd
```
