# lspci Command - Detailed Documentation

## 1. Description
The `lspci` command lists all PCI devices.

## 2. Purpose
The purpose of `lspci` is hardware identification. It is used to:
- Check exactly what graphics card or Wi-Fi card is installed using PCI IDs.

## 3. Examples
### Example 1: PCI Devices
```bash
lspci
```
