# locate Command - Detailed Documentation

## 1. Description
The `locate` command searches a pre-built database for files by name. It is much faster than `find` but can be outdated if the database hasn't been updated.

## 2. Purpose
The purpose of `locate` is rapid search. It is used to:
- Quickly find where a file is.

## 3. Examples
### Example 1: Find File
```bash
locate httpd.conf
```
