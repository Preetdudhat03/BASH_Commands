# fg Command - Detailed Documentation

## 1. Description
The `fg` command brings a background job to the foreground.

## 2. Purpose
The purpose of `fg` is job focus. It is used to:
- Return to a text editor (vim/nano) that was backgrounded.

## 3. Examples
### Example 1: Foreground Last
Bring the most recent background job to front.
```bash
fg
```
