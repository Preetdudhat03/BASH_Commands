# source Command - Detailed Documentation

## 1. Description
The `source` command (often abbreviated as `.`) executes commands from a file in the *current* shell environment.

## 2. Purpose
The purpose of `source` is loading configuration. It is used to:
- Reload `.bashrc` after changes without restarting the terminal.
- Load virtual environment variables.

## 3. Examples
### Example 1: Reload Config
```bash
source ~/.bashrc
```
