# iperf Command - Detailed Documentation

## 1. Description
The `iperf` command performs active measurement of the maximum achievable bandwidth on IP networks.

## 2. Purpose
The purpose of `iperf` is speed testing.

## 3. Examples
### Example 1: Server
```bash
iperf -s
```
### Example 2: Client
```bash
iperf -c 192.168.1.5
```
