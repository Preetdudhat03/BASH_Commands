# tcptraceroute Command - Detailed Documentation

## 1. Description
The `tcptraceroute` command runs a traceroute using TCP packets instead of UDP/ICMP.

## 2. Purpose
The purpose of `tcptraceroute` is tracing routes through firewalls that block standard traceroute.

## 3. Examples
### Example 1: Trace
```bash
sudo tcptraceroute google.com
```
