# swapoff Command - Detailed Documentation

## 1. Description
The `swapoff` command disables devices and files for paging and swapping.

## 2. Purpose
The purpose of `swapoff` is disabling virtual memory. It is used to:
- Turn off swap to resize it or remove the drive.

## 3. Examples
### Example 1: Disable All
```bash
sudo swapoff -a
```
