# iostat Command - Detailed Documentation

## 1. Description
The `iostat` command reports Central Processing Unit (CPU) statistics and input/output statistics for devices and partitions.

## 2. Purpose
The purpose of `iostat` is disk monitoring. It is used to:
- Check if disk I/O is slowing down the server.
- Measure Read/Write speeds.

## 3. Examples
### Example 1: Device Stats
```bash
iostat -d
```
