# md5sum Command - Detailed Documentation

## 1. Description
The `md5sum` command computes and checks MD5 message digests (hashes).

## 2. Purpose
The purpose of `md5sum` is integrity checking. It is used to:
- Verify a downloaded file hasn't been corrupted.
- Check if two files are identical.

## 3. Examples
### Example 1: Generate Hash
```bash
md5sum file.iso
```
