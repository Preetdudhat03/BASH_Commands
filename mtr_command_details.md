# mtr Command - Detailed Documentation

## 1. Description
The `mtr` command (My Traceroute) combines `ping` and `traceroute`. It continuously updates stats for each hop.

## 2. Purpose
The purpose of `mtr` is network diagnostics. It detects packet loss at specific hops over time.

## 3. Examples
### Example 1: Trace
```bash
mtr google.com
```
