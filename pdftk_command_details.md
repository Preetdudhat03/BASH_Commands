# pdftk Command - Detailed Documentation

## 1. Description
The `pdftk` command (PDF Toolkit) is a tool for manipulating PDF documents.

## 2. Purpose
The purpose of `pdftk` is merging, splitting, and rotating PDFs.

## 3. Examples
### Example 1: Merge
```bash
pdftk 1.pdf 2.pdf cat output merged.pdf
```
