# ncdu Command - Detailed Documentation

## 1. Description
The `ncdu` command (NCurses Disk Usage) is a curses-based version of the well-known 'du', and provides a fast way to see what directories are using your disk space.

## 2. Purpose
The purpose of `ncdu` is interactive disk usage analysis and cleanup.

## 3. Examples
### Example 1: Analyze Current Dir
```bash
ncdu
```
