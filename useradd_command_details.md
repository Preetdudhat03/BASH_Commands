# useradd Command - Detailed Documentation

## 1. Description
The `useradd` command creates a new user or updates default new user information.

## 2. Purpose
The purpose of `useradd` is creating accounts. It is used to:
- Create new system users.

## 3. Parameters
- `-m`: Create home directory.
- `-s`: Specific shell.

## 4. Examples
### Example 1: Create User
```bash
sudo useradd -m -s /bin/bash newuser
```
