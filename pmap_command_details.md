# pmap Command - Detailed Documentation

## 1. Description
The `pmap` command reports the memory map of a process or processes. It shows the memory usage of a process in detail.

## 2. Purpose
The purpose of `pmap` is memory analysis.
- Debug memory leaks.
- See how shared libraries are mapped.

## 3. Examples
### Example 1: Show Map
Show memory map of process 1234
```bash
pmap 1234
```
