# cfdisk Command - Detailed Documentation

## 1. Description
The `cfdisk` command is a curses-based program for partitioning any block device. It is more user-friendly than `fdisk`.

## 2. Purpose
The purpose of `cfdisk` is partitioning.

## 3. Examples
### Example 1: Partition
```bash
sudo cfdisk /dev/sdb
```
