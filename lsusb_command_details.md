# lsusb Command - Detailed Documentation

## 1. Description
The `lsusb` command puts a list of USB devices available in the system.

## 2. Purpose
The purpose of `lsusb` is hardware ID. It is used to:
- Check if a USB webcam or drive is detected by the kernel.

## 3. Examples
### Example 1: List USB
```bash
lsusb
```
