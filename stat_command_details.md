# stat Command - Detailed Documentation

## 1. Description
The `stat` command displays detailed file or file system status. It shows much more information than `ls -l`.

## 2. Purpose
The purpose of `stat` is metadata inspection. It is used to:
- See exact access, modify, and change timestamps.
- See inode number and number of blocks.

## 3. Examples
### Example 1: Check File Status
```bash
stat file.txt
```
