# jobs Command - Detailed Documentation

## 1. Description
The `jobs` command displays the status of jobs in the current shell session.

## 2. Purpose
The purpose of `jobs` is session management. It is used to:
- See what tasks are running in the background.
- Find the Job ID to use with `fg` or `bg` (e.g., `%1`).

## 3. Examples
### Example 1: List Jobs
```bash
jobs
```
