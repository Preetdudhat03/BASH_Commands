# xmodmap Command - Detailed Documentation

## 1. Description
The `xmodmap` command is used to edit and display the keyboard modifier map and keymap table.

## 2. Purpose
The purpose of `xmodmap` is modifying keymaps (e.g., swapping Caps Lock and Ctrl).

## 3. Examples
### Example 1: Load Map
```bash
xmodmap ~/.Xmodmap
```
