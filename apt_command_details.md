# apt Command - Detailed Documentation

## 1. Description
The `apt` command provides a high-level commandline interface for the package management system (Debian/Ubuntu). It is intended as an end user interface.

## 2. Purpose
The purpose of `apt` is package management. It combines `apt-get` and `apt-cache`.

## 3. Examples
### Example 1: Install
```bash
sudo apt install git
```
### Example 2: Update
```bash
sudo apt update
```
